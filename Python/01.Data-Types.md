# Data Types and basic methods
### 1) String
#### String: plain texts, use "" to represent
```Shell
character_name = "Dong"
print("he liked the name " + character_name + ".")
```
he liked the name Dong.

#### Basic methods
```Shell
# 1) \n: words behind start appears in a new line
print("Giraffe\nAcademy")

# 2) \": print out a ".
print("Giraffe\"Academy")

# 3) .lower() / .upper(): make strings shows in all lower/upper cases
#   a) .islower(): check if the text is lower
phrase = "Giraffe Academy"
print(phrase.lower())

phrase = "Giraffe Academy"
print(phrase.islower())

phrase = "Giraffe Academy"
print(phrase.upper().isupper())

# 4) len(): to calculate a length of a string
phrase = "Xuzhi Wei"
print(len(phrase))

# 5) [0]: to only show the first letter of a text
phrase = "Giraffe Academy"
print(phrase[4])

# 6) .index(""): tells what position a number or phrase is located
phrase = "Giraffe Academy"
print(phrase.index("G"))

# 7) .replace("a", "b"): replace a with b
phrase = "Giraffe Academy"
print(phrase.replace("Giraffe", "Elephant"))
```

### 2) Number
```Shell
# order of calculation
print(3 * 4 + 5)
print(3 * (4+5))

# modulo operator %: It returns the remainder of dividing the left hand operand by right hand operand.
print(10 % 4)

# str(): convert number into string
my_num = 5
print(str(my_num) + " my favorite number")

# abs (absolute value)
my_num = -5
print(abs(my_num))

# pow(a, b): calculate a square b
print(pow(5, 2))

# max(): show which number is higher
print(max(5, 2))

# round()
print(round(5.2))

# floor():floor(a.b): chop off decimal point b
# ceil():round number up no matter what 
# sqrt()
from math import *
print(floor(3.7))
print(ceil(3.2))
print(sqrt(3.8))
```
